# 🧮 Digital Calculator
Μικρό project για το μάθημα **Τεχνολογίες Λογισμικού** – ΔΙΠΑΕ  
Εβδομάδα 1: Εξάσκηση σε Git, GitHub και βασική Java συνεργασία.

---

## ⚙️ Περιγραφή
Ο στόχος του project είναι να δημιουργηθεί μια **ψηφιακή αριθμομηχανή** που υποστηρίζει:
- Πρόσθεση  
- Αφαίρεση  
- Πολλαπλασιασμό  
- Διαίρεση  

Κάθε μέλος θα υλοποιήσει μια λειτουργία σε ξεχωριστό branch.
